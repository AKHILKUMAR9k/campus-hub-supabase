# **App Name**: Campus Hub

## Core Features:

- User Authentication and Roles: Enable signup/login for students and club organizers with email/Google Auth and role-based permissions for event management and viewing.
- Upcoming Events Page: Display upcoming events with filtering (club/category/date) and search functionalities.
- Event Details Page: Show event details (description, date, venue, registration link) with "Register" button and "Add to Calendar" (Google Calendar/iCal) integration, and a reminder option.
- Past Events Section: Feature a gallery of completed events (images, description) with a comments and suggestions section (like/dislike, replies).
- User Dashboard: Provide students with a registered events list and calendar view. Offer club organizers event creation/editing, media upload, and registration tracking.
- Notifications: Send email/in-app alerts for event reminders, new comments, and registration confirmations.
- Suggested Event Tagging: AI tool that, upon initial event creation, can read the text of the event description, then can suggest descriptive tags. Organizers may accept or reject suggestions.

## Style Guidelines:

- Primary color: Slate blue (#558bbf), conveying trustworthiness.
- Background color: Very light blue (#e5eff5), close in hue to the primary but almost fully desaturated.
- Accent color: Lavender (#9fa4c4), an analogous color providing gentle contrast, with added brightness and saturation.
- Body and headline font: 'PT Sans' (sans-serif) for a modern, accessible feel.
- Use consistent, clean icons from a set like Material Design Icons for navigation and actions.
- Maintain a clean, grid-based layout with consistent spacing and alignment.
- Use subtle animations (e.g., fade-ins, transitions) to enhance user experience.
